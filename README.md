**Простейшее решение для запуска whisper-ai локально.**<br>
**Может транскрибировать и видео, и аудио файлы.**<br>
**Автоматически определяет язык, но есть возможность указывать его самостоятельно (в т.ч. с переводом), все подробности в официальном репозитории, указанном ниже.**<br>
<br>
**Версия python3.9**<br>
**Обязательно скачайте ffmpeg по ссылке:** https://www.ffmpeg.org/download.html<br>
**Ссылка на репозиторий whisper-ai:** https://github.com/openai/whisper<br>
