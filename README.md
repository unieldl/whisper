Простейшее решение для запуска whisper-ai локально.
Может транскрибировать и видео, и аудио файлы
Автоматически определяет язык, но есть возможность указывать его самостоятельно (в т.ч. с переводом), все подробности в официальном репозитории, указанном ниже

Версия python3.9
Обязательно скачайте ffmpeg по ссылке: https://www.ffmpeg.org/download.html
Ссылка на репозиторий whisper-ai: https://github.com/openai/whisper
