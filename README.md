Простейшее решение для запуска whisper-ai локально.\n
Может транскрибировать и видео, и аудио файлы.\n
Автоматически определяет язык, но есть возможность указывать его самостоятельно (в т.ч. с переводом), все подробности в официальном репозитории, указанном ниже.\n
\n
Версия python3.9\n
Обязательно скачайте ffmpeg по ссылке: https://www.ffmpeg.org/download.html\n
Ссылка на репозиторий whisper-ai: https://github.com/openai/whisper\n
